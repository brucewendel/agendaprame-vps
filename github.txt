===============================
 GUIA: SUBIR CÓDIGO NO GITHUB
===============================

# 1. Acesse sua pasta do projeto
cd /caminho/para/sua/pasta

# 2. Inicie o Git (se ainda não iniciou)
git init

# 3. Adicione o repositório remoto (só se ainda não fez)
git remote add origin https://github.com/brucewendel/AGENDAMENTO_SALAS

# 4. Adicione os arquivos para commit
git add .

# 5. Faça o commit
git commit -m "Mensagem do commit"

# 6. Suba para o GitHub
git push -u origin main

# ⚠️ Se 'main' não existir, tente:
git push -u origin master

# Comandos úteis:
git status       # Ver o que mudou
git branch       # Ver em qual branch você está
git remote -v    # Ver repositório remoto

==============================
 COMO USAR O .GITIGNORE
==============================

# 1. Crie o arquivo na raiz do projeto
touch .gitignore

# 2. Adicione os arquivos/pastas que deseja ignorar:

Exemplo de conteúdo:

# Ambiente virtual
venv/

# Variáveis de ambiente
.env

# Arquivos Python compilados
__pycache__/
*.pyc

# Arquivos temporários de sistema
.DS_Store
Thumbs.db

# Logs
*.log

# 3. Se você já adicionou um arquivo que quer ignorar:
git rm --cached NOME_DO_ARQUIVO

Ex:
git rm --cached .env



# Depois disso:
git add .
git commit -m "Atualiza .gitignore"
git push
